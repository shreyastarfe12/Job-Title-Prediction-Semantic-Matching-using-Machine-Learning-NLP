# Job-Title-Prediction-Semantic-Matching-using-Machine-Learning-NLP
This project involves an in-depth analysis of job description data with a primary focus on salary 
information. The dataset includes salary figures expressed as ranges, which were preprocessed to 
isolate minimum and maximum values. These were then converted into numerical formats and 
used to compute an average salary for each job posting, enabling standardized comparisons 
across roles. 
The analysis began with thorough data preparation, including identifying missing values and 
ensuring consistency in salary formats. By splitting and transforming the "Salary Range" 
column, meaningful salary insights were extracted that could be used in further analysis. 
The key objective of this project is to gain insights into compensation patterns across various job 
listings. These findings can support job seekers, HR departments, and market researchers by 
providing clear benchmarks for salaries. Python libraries such as Pandas and Scikit-learn were 
used for data processing and scaling, laying the groundwork for future exploration, including 
potential predictive salary models. 
This project demonstrates the ability to clean and transform semi-structured data, perform 
foundational analysis, and derive insights with practical relevance in the job market landscape.
